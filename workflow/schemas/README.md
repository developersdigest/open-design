# Brand API JSON Schemas

JSON Schema (draft 2020-12) definitions for the 4 brand webhook routes.

| File | Description |
| --- | --- |
| `decode.request.schema.json` | Request body for `POST /webhook/brand/decode`. |
| `decode.response.schema.json` | Response body for `POST /webhook/brand/decode`. |
| `design.request.schema.json` | Request body for `POST /webhook/brand/design`. |
| `design.response.schema.json` | Response body for `POST /webhook/brand/design`. |
| `html.request.schema.json` | Request body for `POST /webhook/brand/html`. |
| `html.response.schema.json` | Response body for `POST /webhook/brand/html`. |
| `assets.request.schema.json` | Request body for `POST /webhook/brand/assets`. |
| `assets.response.schema.json` | Response body for `POST /webhook/brand/assets`. |

See `../README.md` for route contracts and `../../examples/` for example payloads.
