# Developers Digest Open Design

Open-source, MIT-licensed full-stack template. Paste any URL and turn it into a brand kit, a design system, reusable CSS, HTML components, and image assets — powered by a single self-hosted **n8n** workflow behind a real **Next.js 16** app.

Most n8n demos show a workflow in isolation. This shows n8n as the **backend of a real product** — front end, UX, and all.
